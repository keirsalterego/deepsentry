Model architecture

The project combines a text autoencoder with a recurrent anomaly detector.

StagePurpose
Text autoencoderCompresses log messages into a learned latent representation
Sequence modelScores encoded sequences for unusual reconstruction behavior
Live detectorApplies rolling anomaly checks to incoming log lines

Code areas

  • ta/ contains the text autoencoder package
  • kad/ contains the recurrent anomaly model package
  • src/tx combines encoding and dataset preparation
  • src/an handles scoring and analysis
