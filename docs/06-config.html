Configuration

Configuration files live under dockerconfig and are loaded through src/config_loader.py.

Available config files

  • prepare.yml
  • train.yml
  • encode.yml
  • anomaly_train.yml
  • anomaly_eval.yml
  • anomaly_eval_labeled.yml
  • live.yml
  • logging_config.yml

The loader replaces {ROOT_DATA_DIR} with the path you pass at runtime, so the same configs can work in Docker and local setups.
