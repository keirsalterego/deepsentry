Pipeline

The pipeline is staged, with each step writing output that feeds the next step.

  1. Prepare raw logs for text training.
  2. Train the text autoencoder.
  3. Encode the full dataset into latent vectors.
  4. Train the anomaly detector.
  5. Evaluate anomaly scores.
  6. Run the labeled analysis and live monitor.

Run order

sh dockerrun/run_prepare.sh /path/to/data /root/data
sh dockerrun/run_text.sh /path/to/data /root/data
sh dockerrun/run_encode.sh /path/to/data /root/data
sh dockerrun/run_anomaly_train.sh /path/to/data /root/data
sh dockerrun/run_anomaly_eval.sh /path/to/data /root/data
