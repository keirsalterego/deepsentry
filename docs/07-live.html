Live monitoring

The live loop streams log lines, encodes them, scores them, and prints anomalies with a running threshold.

Important behavior

  • Time based batching uses a configurable interval
  • Encoded lines are normalized with a cosine time transform
  • Rolling stats are used to adjust the anomaly threshold

The live code is still a good place for future production hardening, especially around input validation and graceful shutdown handling.
