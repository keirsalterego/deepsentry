Data and files

The project expects a data root folder with a dockerconfig subfolder and output folders created by the pipeline.

Important paths

  • dockerconfig/ stores runtime YAML files
  • prepared_data/ stores train and validation text data
  • encoded_dataset/ stores encoded pickles
  • anomaly_trained_model/ stores the anomaly model
  • text_autoencoder_model/ stores the text model

The configuration loader replaces the {ROOT_DATA_DIR} placeholder with the path you pass on the command line.
