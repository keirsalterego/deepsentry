What this project does

DeepSentry trains a text autoencoder on raw log messages, encodes the resulting sequences, then scores anomalies using a recurrent autoencoder model.

What is already implemented

  • Log preprocessing and tokenization
  • Text autoencoder training
  • Dataset encoding and anomaly scoring
  • Labeled evaluation and plots
  • Live monitoring loop
  • Docker based execution

What still remains for a production grade release

  • Broader tests
  • More validation on inputs and config values
  • A proper CLI wrapper
  • Packaging metadata
  • Continuous integration beyond the smoke test
