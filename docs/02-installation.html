Installation

The project is designed to run inside Docker, with a local Python environment available for tests and docs.

ItemPurpose
DockerPrimary runtime for training and evaluation
Python venvLocal smoke tests and doc maintenance
CUDA GPUUseful for larger training jobs

Quick start

./build.sh .
sh dockerrun/run_prepare.sh /path/to/data /root/data
sh dockerrun/run_text.sh /path/to/data /root/data

The next chapter explains the expected data folder layout.
