Introduction

The project learns a baseline of normal behavior from raw log data and flags deviations as potential anomalies.

The repository implements the experiment pipeline from the original paper and repackages it with shorter names and a cleaner documentation surface.

Core idea

Instead of trying to teach the system every attack, the model learns what normal looks like. Anything outside that learned pattern is treated as suspicious.

Important modules

  • src/raw_log_file_process.py parses log lines
  • src/tx handles text encoding stages
  • src/an handles anomaly scoring and evaluation
  • src/live/main.py runs streaming monitoring

The next chapter shows how to set up the environment and run the project.
